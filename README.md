# Backend Cursos

Projeto desenvolvido para a disciplina Programação para Internet.

## Tecnologias
- Node.js
- Express
- MySQL

## Funcionalidades
- Cadastro de cursos (POST)
- Listagem de cursos (GET)
- Atualização de cursos (PUT)
- Exclusão de cursos (DELETE)

## Como rodar
npm install
node server.js
